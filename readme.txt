Рабочий скрипт:
convert.py

Принимает 2 обязательных параметра строки:
входной файл и выходной формат (вызываемый плагин)

При запуске с неверным числом параметров выдаёт подсказку по использованию.

Для запуска всех 4-х тестов с заданным файлом служит никсовый скрипт
test_run.sh

Использовались только стандартные библиотеки.

Реализация плагинов далека от совершенства - просто и без посторонних библиотек.

Плагин должен располагаться в каталоге plugins, содержать класс, унаследованный от base.baseplugin и реализующий run и write
Run возвращает имя и описание плагина, которые используются в выводе помощи (usage) и при выборе конкретного плагина.

В базовом классе реализован метод translate, переводящий заранее заданные слова с русского на английский. Минимальная реализация в рамках ТЗ.
